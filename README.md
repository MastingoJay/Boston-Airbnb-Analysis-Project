# Boston Airbnb Market & Performance Analysis - Boston,USA

![](https://github.com/MastingoJay/Boston-Airbnb-Analysis-Project/blob/main/boston.jfif)

# Overview

This project delivers a full end-to-end data analytics workflow to help Airbnb understand host success, neighborhood market conditions, and guest experience quality across the Boston market.
It moves beyond simple EDA and provides actionable insights that directly answer key business questions.

# Business Questions

### 1. Host Success & Market Dynamics

- What differentiates successful hosts from struggling ones?

- How can Airbnb help more hosts succeed?

### 2. Neighborhood Impact & Market Saturation

- Which Boston neighborhoods show strong demand vs. oversaturation?

- What does this mean for Airbnb’s growth strategy?

### 3. Guest Experience & Quality Assurance

- What drives exceptional guest experiences?

- How can Airbnb detect quality problems early?

# Tools & Technologies Used

1. Python: For web scraping and data processing

2. PostgreSQL: For structured data storage and querying

3. Power BI: For data cleaning, modeling, and dashboard visualization

# Project Workflow

### 1. Data Preparation in Python

A. Loaded raw Boston Airbnb datasets

B. Validated structure & completeness

C. Treated missing values and outliers

D. Removed duplicates

E. Built univariate & bivariate EDA:

- Pricing trends

- Room type performance

- Listing distribution

- Neighborhood-level patterns

- Correlation heatmaps

### 2. Database Modeling in PostgreSQL

1. Designed database schema

2. Loaded cleaned data with psycopg2

3. Created analytical SQL queries & CTEs:

- Host success metrics

- Market health scoring

- Neighborhood saturation analysis

- Guest engagement & review velocity

### 3. Power BI Dashboard Development

1. Connected PostgreSQL to Power BI

2. Created KPIs using DAX

3. Designed interactive insights dashboards:

- Home – Landing page with interactive navigation buttons

- Host performance - Core business visuals with slicers and insights

- Neighborhood market map

- Guest experience quality tracker

- Insights – Written interpretations of data trends

- Added slicers (price,availability,room price) for dynamic filtering
